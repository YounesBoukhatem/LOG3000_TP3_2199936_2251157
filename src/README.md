## src
Ce module contient la logique de calcul de l'application.

## Fichiers
- operators.py: contient les fonctions correspondantes aux opérations arithmétiques exécutées par la calculatrice
    - Addition
    - Soustraction
    - Multiplication
    - Division
- app.py: contient la logique de calcul et le routage des requêtes HTTP.

## Dépendances
app.py utilise les fonctions d'opérations arithmétiques contenues dans operators.py.
